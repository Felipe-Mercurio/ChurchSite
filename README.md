Instalar (next + react + react-dom):
    npm install next react react-dom

Instalar Babel:
    npm install --save-dev babel-loader @babel/core @babel/preset-react

Configurar Git:
    git add -A
    git commit -m 'First page'
    git push origin main

Para rodar o projeto:
    npm run dev

Para para o projeto:
    Ctrl + C
